# InternView

InternView — это iOS-приложение для поиска стажировок и управления вакансиями. Проект реализован на **SwiftUI** и использует **Basic Authentication** для связи с сервером, написанным на **Vapor** (Swift). Приложение позволяет интернам создавать резюме, откликаться на вакансии, а рекрутерам — публиковать вакансии и просматривать отклики.

---

## 🚀 Функциональность

### 👨‍💻 Для интернов
- Регистрация и вход
- Просмотр активных вакансий с фильтрацией (формат работы, уровень)
- Создание и редактирование резюме (CV) с загрузкой PDF-файла
- Отклик на вакансию с выбором резюме и сопроводительным письмом
- Просмотр истории своих откликов и их статусов
- Редактирование профиля

### 👩‍💼 Для рекрутеров
- Регистрация и вход
- Создание, редактирование и удаление вакансий
- Просмотр списка своих вакансий с количеством откликов
- Просмотр всех откликов на конкретную вакансию
- Изменение статуса отклика (новый, рассмотрено, одобрено, отклонено)
- Редактирование профиля

---

## 🛠 Технологии

- **iOS**: SwiftUI, SwiftData (кэширование), Keychain (безопасное хранение учетных данных)
- **Сеть**: URLSession, Basic Auth, multipart/form-data (загрузка файлов)
- **Архитектура**: MVVM с использованием `@StateObject` и `@Published`

## 📦 Установка бекенда (Vapor)

Для работы приложения необходимо запустить серверную часть. Проект бекенда находится в отдельном репозитории и поставляется с готовым **Docker Compose**.

### 1. Клонировать репозиторий бекенда

```bash
git clone https://github.com/oartemiy/InternView_backend.git
cd InternView_backend
```

---

## 🐳 Запуск через Docker Compose

### 3. Запустить контейнеры

```bash
docker-compose --build
docker-compose up -d
```
